# JaineePatel-2505654-GlobeQuesthotels
This project is an interactive prototype created using Figma to demonstrate a personalized “Culture Immersion” hotel experience.

Description:
This project is an interactive prototype created using Figma to demonstrate a personalized “Culture Immersion” hotel experience. The prototype showcases how users can input their interests such as food, art, and history, and receive AI-driven recommendations tailored to their preferences.
The design follows a user-centered approach, focusing on creating an engaging and intuitive journey from onboarding to experiencing cultural activities. The prototype includes key features such as interest selection, AI-powered suggestions, and immersive experiences like virtual tours and smart room adaptations.
The prototype was tested with users to evaluate usability and personalization. Based on feedback, improvements were made to enhance clarity, navigation, and overall user experience. Prototyping helps visualize user flow and interactions before final development, making it easier to refine ideas and test concepts effectively

User Persona:
Name: Alex (Traveler) Age: 25–40 Type: International traveler
Needs:
Wants authentic cultural experiences Prefers personalized recommendations Enjoys food, art, and history
Pain Points-
Generic hotel experiences Lack of local cultural connection Difficulty planning activities

Research & Insights:
Users prefer personalized experiences over generic services
Cultural engagement increases emotional connection with travel
Simple and clear interfaces improve usability
AI recommendations make decision-making easier

Key Features:
Interest Selection System
Users choose preferences like food, art, and history
AI Personalization
The system suggests customized cultural activities
Interactive Experience
Includes virtual tours, smart room adaptation, and cultural storytelling
Simple User Interface
Easy navigation and clean design

User Flow:
User opens the app (Welcome Screen)
User selects interests (Food, Art, History)
AI generates personalized recommendations
User explores cultural experiences (VR / Smart Room)
User gives feedback

Conclusion:
This prototype successfully demonstrates how AI can transform hotel stays into immersive cultural experiences. The project highlights the importance of user-centered design, personalization, and iterative improvement through testing.
